    Le camp de Guantánamo est un centre de détention militaire de haute sécurité situé sur la base navale américaine, dans le sud-est de Cuba. Vous venez d'y être transférés, les cellules sont séparées, pour être sûr que personne n'arrive à en sortir. Mais vous êtes malin, dans chaque endroit hautement sécurisé se trouve toujours une faille... Ici, c'est un des gardiens, il se lie auprès de nombreux prisonniers pour ce faire ses pourboires. Ce gardien chargé de votre transfert semble intéressé, il vous aidera si vous trouvez de quoi le payer !
