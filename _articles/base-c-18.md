    22 Septembre 1944, Jean Morel, lieutenant colonel des forces secrètes aériennes françaises est pris au piège dans son bunker qui vient d’être assiégé par l’ennemi. Plusieurs informations classées « Secret Défense » sont dévoilées, dont l’emplacement d’un lieu tactique confidentiel : la Base C-18. Cette base aérienne secrète abrite de nombreux avions de combats mais va être bombardée par l’ennemi d’ici peu de temps. Vous, Agents TimeXperience, allez être transportés dans la Base C-18 quelques heures avant son bombardement et la destruction totale de ce lieu tactique. Il se dit que cette Base militaire renferme de nombreux secrets de guerre conservés dans un dossier confidentiel.
